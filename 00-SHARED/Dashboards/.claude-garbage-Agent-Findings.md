# Agent Findings — Reviewer Scratchpad

Use this dashboard to see **recently updated investigations and reports** where agents may have added questions, discoveries, or connections. **Click a link** to open the note and get:

- **Evidence trail** — Evidence Chain / OSINT Sources in the same note  
- **State of investigation** — What’s been answered, what’s open  
- **Agent questions (for reviewer)** — Open questions for you to answer or triage  
- **Interesting discoveries / scratchpad** — Mini-findings and “might be relevant” notes  
- **New questions raised** — Follow-up questions and new threads  
- **Data needed to answer** — What data or sources would close gaps  
- **Connections to other work** — Links to related investigations, entities, evidence  

Agents should write into those sections as they work so nothing is lost and you can thread findings into other work.

---

## Recently updated (click for full note + evidence trail)

Investigations and reports, most recently modified first. Open any row to see agent scratchpad sections and evidence.

```dataview
TABLE
  file.link as "Investigation / report",
  type as "Type",
  dateformat(file.mtime, "yyyy-MM-dd HH:mm") as "Last updated"
FROM "00-SHARED"
SORT file.mtime DESC
LIMIT 30
```

## If you use a separate Reports folder

Add a block like this and change the path to your reports folder (e.g. `50-Reports`):

```dataview
TABLE file.link as "Report", dateformat(file.mtime, "yyyy-MM-dd HH:mm") as "Last updated"
FROM "50-Reports"
WHERE type = "report"
SORT file.mtime DESC
LIMIT 20
```
