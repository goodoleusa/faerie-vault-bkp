# Investigation Overview

## Active Investigations
```dataview
TABLE status, threat_level, last_update, deadline
FROM "10-Investigations" OR "00-SHARED/01-MEMORY-EDIT"
WHERE status = "active"
SORT threat_level ASC, deadline ASC
```

## Recent Entity Updates
```dataview
TABLE entity_type, confidence_level, priority, status
FROM "20-Entities"
SORT file.mtime DESC
LIMIT 20
```

## Evidence Pipeline
```dataview
TABLE tier, confidence_level, hypothesis_support, pipeline_run
FROM "30-Evidence"
SORT tier ASC, confidence_level DESC
LIMIT 30
```

## Pending Tasks
```dataview
TABLE priority, assignee, deadline, status
FROM "00-Inbox"
WHERE type = "task" AND status != "done"
SORT priority ASC, deadline ASC
```

## Quick Actions

Use **Command Palette** → **QuickAdd** → pick one (creates note in the right folder; then run **Blueprint: Apply blueprint to note** to fill placeholders).

| Action | QuickAdd choice | Folder | Notes |
|--------|-----------------|--------|-------|
| New Task | **New Task** | 00-Inbox | Prompt: task title |
| New Person | **New Person** | 20-Entities/People | Prompt: person name |
| New Investigation | **New Investigation** | 10-Investigations | Prompt: case name |
| New Entity | **New Entity Stub** | 20-Entities | Prompt: entity name |
| New Evidence | **New Evidence Item** | 30-Evidence | Prompt: evidence ID |
| New Source | **New Source** | 70-Sources | Prompt: source name |
| Today's research | **Today's Research Log** | 01-Memories | Prompt: agent name → `YYYY-MM-DD-Research-AgentName.md` |

See [[ARCHITECTURE#12. QuickAdd + Blueprints (Obsidian)]] for macros (create + apply blueprint in one step) and Blueprint command tips.
