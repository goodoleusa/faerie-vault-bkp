## How This Works

```
Session ends → session_stop_hook.py fires → session_manifest.py runs
  ├─ Scans all workspace repos for recent commits
  ├─ Collects HONEY entries added this session
  ├─ Lists memory files touched
  ├─ Writes JSON to ~/.claude/hooks/state/session-manifests/
  └─ Writes Markdown to vault 00-SHARED/Dashboards/session-manifests/
```

The manifest answers: "What happened this session?" across all repos simultaneously.
The brief answers: "What is the investigation state?" at session end.
Together they give the full picture.
