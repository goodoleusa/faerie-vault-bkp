# Mission Control — Cross-Session Intelligence

> *What every session should see. Agent activity, live drops, annotation queue, cross-session connections.*
> *Refresh: `Ctrl+Shift+R`*

---

## ⚡ Live — Agent Activity

```dataview
TABLE status, priority, skill_required, assignee, file.mtime AS "Updated"
FROM "00-Inbox/tasks"
WHERE status = "claimed" OR status = "in-progress"
SORT file.mtime DESC
LIMIT 8
```

---

## 📬 Unread Drops (Last 24h)

```dataview
TABLE drop_type, from, priority, file.ctime AS "Received"
FROM "00-Inbox/drops"
WHERE read = false
SORT priority ASC, file.ctime DESC
```

---

## 🔴 Annotate These — Tier 1 Unannotated

```dataview
TABLE evidence_id, confidence_level, source_type, date_collected, chain_id
FROM "30-Evidence" OR "00-SHARED/Agent-Outbox"
WHERE tier = 1 AND (ann_hash = null OR ann_hash = "")
SORT confidence_level DESC
LIMIT 8
```

---

## 🆕 Agent Deliveries — Awaiting Review

```dataview
TABLE evidence_id, confidence_level, tier, pipeline_run, agent_author, file.ctime AS "Delivered"
FROM "00-SHARED/Agent-Outbox"
WHERE status = "awaiting-human"
SORT file.ctime DESC
LIMIT 10
```

## 🔗 Cross-Session Connections (Last 7 Days)

```dataview
TABLE file.link, hypothesis_support, confidence_level, promotion_state, file.mtime AS "Found"
FROM ""
WHERE type = "evidence" AND file.mtime >= date(today) - dur(7 days)
  AND promotion_state != "archived"
SORT confidence_level DESC
LIMIT 10
```

## ⏳ Threads — Your Turn

```dataview
TABLE thread_title, started_by, updated
FROM ""
WHERE type = "thread" AND contains(needs_response_from, "human")
SORT updated DESC
LIMIT 6
```

---

## 🧠 Memory Layer — Ready to Promote

```dataview
TABLE type, memory_lane, file.mtime AS "Updated"
FROM ""
WHERE promotion_state = "capture"
  AND (memory_lane = "annotation-queue" OR type = "evidence")
  AND ann_hash != null AND ann_hash != ""
SORT file.mtime DESC
LIMIT 8
```

## 📊 Evidence Pipeline — Session Overview

```dataview
TABLE WITHOUT ID
  rows.promotion_state AS "Stage",
  length(rows) AS "Count"
FROM "30-Evidence" OR "00-SHARED/Agent-Outbox"
GROUP BY promotion_state
```

TABLE session_id, turns, model, ended_at
