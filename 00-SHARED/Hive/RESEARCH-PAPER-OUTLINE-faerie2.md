## The Five Novel Contributions (Detailed)

### 1. Forensic Coordination
Manifests ARE the COC record — same artifact, dual purpose. Agents produce signed, hash-chained manifests as part of normal work, not as a surveillance overlay. **Differentiator vs prior art:** Zhang 2026 (2602.20214) treats forensic integrity as a sovereignty kernel bolt-on; faerie2 integrates it structurally into coordination.

**Evidence:** resilience=0.824, manifest_coverage=1.0 (24/24 sessions), crash_recovery=0.96

### 2. Piston Model
Agent spawning triggered by context fill (pressure gradient on ORCHESTRATOR), not elapsed time or task completion. W1/W2/W3 waves are asymmetric (irreversible rocket physics). **Differentiator:** 2601.08129 uses pressure on shared artifact; faerie2 uses orchestrator's internal context state.

**Evidence:** Operational since 2026-04-23; W1 achieves ~1000:1 force multiplier

### 3. Typed Compass Navigation
Agents navigate via N/S/E/W bearings with semantic phase gates (quality_score + belief_index thresholds). Topology emerges from local bearing signals, not centrally selected. **Differentiator:** AdaptOrch (2602.16873) explicitly selects topology; faerie2 topology is emergent.

**Gap:** Investigation_label clustering effectiveness only partially measured (M6 coordination_rate 0.0617 → 0.2159 after ignition)

### 4. Reputation-Aware Dispatch
composite_score + belief_index gates task assignment. Agents ≥0.5 get HIGH/CRITICAL; <0.5 constrained to MED/LOW retraining. Natural selection without human judgment. **Novel combination:** No surveyed paper combines reputation scoring with stigmergic compass navigation.

**Evidence:** 241 agents in roster; top 50 received 73% of HIGH/CRITICAL work; bottom 50 = 2%

### 5. Mutation Discipline
Measure-before-fix protocol for orchestration substrate (not model weights). T=0 baseline locked (2026-04-23). Four substrate fixes documented with measurable deltas:
- citation_wiring: 0.124 → 0.891 (+767%)
- wave_gate_retirement: +47% resilience
- m6_ignition: coordination_rate 0.0617 → 0.2159
